# 🧪 Interactive Periodic Table

A responsive and interactive periodic table built using HTML, CSS and JavaScript.

## ✨ Features

- 118 Elements
- Search functionality
- Element details popup
- Dark theme UI
- Responsive layout

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
